React creates a virtual DOM. When state changes in a component it firstly runs a "diffing" algorithm, which identifies what has changed in the virtual DOM. The second step is reconciliation, where it updates the DOM with the results of diff. react creates a virtual dom // real dom copy shadow dom // web components virtual dom // react state // data // !
