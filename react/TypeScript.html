*** TypeScript *** TypeScript Anders Hejlsberg tarafindan olsuturldu TypeScript, JavaScript'in ustune gelistirilmis bir dildir. JavaScript'in yazilim hatalarini ve kodlama hatalarini onlem almak icin kullanilir. Installation npm install -g typescript tsc --version tsc --init tsc -w Root Directory for typescript src out directory for out Type annotation bir degiskenin typini belirlemek icin kullanilir ornegin: let a: number = 5;
